# Vocabulary Trainer

Современное приложение для пополнения словарного запаса путем изучения новых слов с помощью интерактивных тестов.

## Возможности

### 🎯 Основной функционал
- **Несколько режимов тестирования**:
  - Английский → Русский (`en_to_ru`)
  - Русский → Английский (`ru_to_en`)
  - Смешанный режим (`mixed`)
- **Настраиваемые тесты**: Выбор количества слов и уровня сложности
- **Управление словарем**: Добавление, удаление и импорт слов
- **Отслеживание прогресса**: Подробная статистика и аналитика

### 📊 Уровни сложности
- **Легкий уровень**: Без ограничения времени, с подсказками (первая буква)
- **Средний уровень**: Ограничение 30 секунд, без подсказок
- **Сложный уровень**: Ограничение 15 секунд, без подсказок

### 💾 Управление данными
- **Сохранение прогресса**: Словарь и результаты тестов
- **Импорт слов**: Загрузка списков слов из файлов
